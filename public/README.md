# Public Folder

Web assets for the Airspace Live Visualizer.

## Running the Demo
Serve this folder with a static server:

```bash
npx serve . -l 8000
# or
python -m http.server 8000
```

Open <http://localhost:8000> in a browser.

## Prerequisites
Modern browser with ES module support.
